## highlights вңЁ

**рҹ”¬ lifecycle hook visualizer** вҖ” real-time log showing exactly which hook fires, when, and why. built with `runOutsideAngular` + buffer/flush pattern to dodge infinite CD loops.

**рҹ“қ signal forms** вҖ” working demo of Angular 21's `@angular/forms/signals` API with full CRUD + validation. barely anyone's built this yet.

**вҡЎ fully zoneless** вҖ” zero Zone.js. entire app runs on `provideExperimentalZonelessChangeDetection()`.
